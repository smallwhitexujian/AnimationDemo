# demo_animation

Flutter动画插件，

## Getting Started

使用方式比较简单，只需设定起始坐标，设置需要改变的空间即可.
```Dart
    AudioAllMicGift(
        endOffset:RandomScreenOffset.generateRandomScreenCoordinates(context),
        durationTime: 3000,
        offSetY: 100,
        childWidget: const Icon(Icons.accessibility),
```
